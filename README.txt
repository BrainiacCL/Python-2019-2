- Debemos instalar python (https://www.python.org/downloads/release/python-382/).
- Instalamos cmder (https://cmder.net/).
- Instalamos pip (https://pip.pypa.io/en/stable/installing/).
- Abrimos cmder, ejecutando como administrador, e ingresamos, a traves de la consola, 
  a la carpeta donde tenemos nuestro proyecto.
- Cuando en cmder, estemos en la carpeta donde esta nuestro proyecto, ejecutamos el comando
  pip install -r requirements.txt, de esta manera se instalara todo lo necesario para poder ver nuestro sistema.
- Luego, en nuestra consola(cmder), ejecutamos el comando vacunas.py para "prender" el servidor y ejecutar nuestro codigo.
- Abrimos nuestro navegador web y en la URL ingresamos http://127.0.0.1:5000/
- Nos mostrará la pantalla principal de nuestra pagina web.
